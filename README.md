# İlk Türk Devletleri · Eşini Bul

RTF notlarındaki 150 bilgiyi tanım ve cevap kartlarıyla eşleştiren, bağımlılıksız ve mobil uyumlu web oyunu.

Eşleşen kartlar aktif desteden çıkarılır; aynı kart ancak deste baştan karıştırılırsa tekrar görünür.

## Çalıştırma

`index.html` dosyasını doğrudan tarayıcıda açabilir veya klasörde basit bir yerel sunucu başlatabilirsiniz:

```bash
python3 -m http.server 8000
```

Ardından `http://localhost:8000` adresini açın.
